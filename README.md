## Jekyll, Webpack, Vue

**INSTALL**

```sh
npm install
sudo gem install jekyll bundler
sudo gem update --system
bundle install --path vendor/bundle
```

**DEV**

```
npm run dev
```

**BUILD**

```
npm run build
```
(Site build is in ```/dist``` folder)

**SERVE**
```
npm run serve
```

**BUILD & SERVE**
```
npm run build:serve
```
